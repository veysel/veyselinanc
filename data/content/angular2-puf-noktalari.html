Angular 2 Püf Noktaları

Angular 2 için kullanabileceğiniz işinizi kolaylaştıracak kısa önerileri bu yazıda bulabilirsiniz.

Dizi içerisinde gezme işlemleri

Normal Yöntem
let array = [1,2,3,4,5];
for(let i = 0; i < array.length; i++)
{
    console.log(array[i]);
}

Pratik Yöntem
let array = [1,2,3,4,5];
array.forEach(member => {
    console.log(member);
});


Dizi aktarma işlemleri

Normal Yöntem
let array = [1,2,3,4,5];
let tempArray = [];
for(let i = 0; i < array.length; i++)
{
    tempArray.push(array[i]);
}

Pratik Yöntem
let array = [1,2,3,4,5];
let tempArray = array.map(member => {
    return member;
});


Dizi filtreleme işlemleri

Normal Yöntem
let array = [1,2,3,4,5];
let tempArray = [];
for(let i = 0; i < array.length; i++)
{
    if(array[i] > 3)
    {
        tempArray.push(array[i]);
    }
}

Pratik Yöntem
let array = [1,2,3,4,5];
let tempArray = array.filter(member => { member > 3; });
